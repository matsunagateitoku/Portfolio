## About

I build analyst-facing tools at the intersection of **translation technology**, **NLP**, and **AI integration**. My work focuses on bridging machine learning capabilities with real analyst workflows — from federated glossary search to multi-model evaluation pipelines.

Previously a visiting professor in East Asian Studies. Fluent in the gap between what language models do and what translators actually need.
